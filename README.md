# 我的 doom-emacs 配置

安装方法：

1. 安装 emacs-plus

```shell
brew tap d12frosted/emacs-plus
brew install emacs-plus
```

2. 安装 doom-emacs

```shell
git clone https://github.com/hlissner/doom-emacs ~/.emacs.d
```

3. 在用户 HOME 目录下执行

```shell
git clone git@github.com:cnsunyour/.doom.d.git ~/.doom.d
```

4. 执行 doom 更新检查脚本

```shell
~/.emacs.d/bin/doom re
```

5. 启动 Emacs
